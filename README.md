# remove-corrupted-files

Remover arquivos corrompidos.

## Dependências

- Python v3.11.0
- pip v23.1.2

## Como rodar?

- Instale os pacotes necessários

```sh
pip install -r requirements.txt
```

- Rodar a aplicação

```sh
python main.py
```

### Parâmetros adicionais (flags)

- `-d` ou `--dirpath`
